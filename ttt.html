HTML5 (Hypertext Markup Language):
  • HTML is the standard markup language used to create the structure and content of web pages.
  • HTML5 is the latest version of HTML, and it introduces new features and improvements over its predecessors.
  • It provides tags and attributes that define the various elements on a web page, such as headings, paragraphs, images, links, forms, and more.
  • HTML5 also includes new elements like header, nav, article, section, footer,which help in structuring the content more semantically.

Example of a simple HTML5 document structure:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My First Web Page</title> </head> <body> <header> <h1>Welcome to My Website</h1> </header> <nav> <ul> <li><a href="#home">Home</a></li> <li><a href="#about">About</a></li> <li><a href="#contact">Contact</a></li> </ul> </nav> <section> <h2>About Us</h2> <p>This is a brief description of our website...</p> </section> <footer> <p>© 2024 My Website. All rights reserved.</p> </footer> </body> </html>
CSS (Cascading Style Sheets):
  • CSS is used to style and layout the visual presentation of HTML documents.
  • It allows you to control the colors, fonts, spacing, and positioning of elements on a web page.
  • CSS is applied either internally within an HTML document, externally in a separate stylesheet file, or inline within HTML tags.

Example of a simple CSS style rule:

body { font-family: 'Arial', sans-serif; background-color: #f4f4f4; } h1 { color: #333; } nav ul { list-style: none; margin: 0; padding: 0; } footer { background-color: #333; color: #fff; text-align: center; padding: 10px; }
JavaScript:
  • JavaScript is a scripting language that enables dynamic and interactive behavior on web pages.
  • It can manipulate the content and structure of HTML, handle user input, and communicate with servers (Ajax).
  • JavaScript is commonly used to create features like sliders, form validation, pop-up messages, and more.
  • It can be included directly within HTML documents or linked externally.

Example of a simple JavaScript script:

// Simple alert when the page loads window.onload = function() { alert("Welcome to our website!"); } // Example of handling a button click function displayMessage() { var message = "Button clicked!"; console.log(message); alert(message); }

WSL setup

Windows Subsystem for Linux (WSL) is a feature of Windows that allows you to run a Linux environment on your Windows machine, without the need for a separate virtual machine or dual booting. WSL is designed to provide a seamless and productive experience for developers who want to use both Windows and Linux at the same time.

List of Command

wsl --install

installs Linux on you computer

wsl

opens Linux in your terminal

clear

clears the terminal screen

Apache Setup

List of Apache Command

sudo apt update

updates Linux before installation of Apache

sudo apt upgrade

upgrades Linux

sudo apt-get install apache2

sudo service apache2 start

sudo service apache2 stop

stops Apache Server
